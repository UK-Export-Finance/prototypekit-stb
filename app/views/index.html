Use the Clear data link in the footer before starting the journeys again and to get back to this screen click on the UK Export Finance logo in the header.

Version 1

To complete the reconciliation tasks follow these steps:

  1. Complete the reconciliation process for Barclays assuming that the reported fee has been recieved.
  2. Find the Barclays report from April 2023.
  3. Complete the reconciliation process for the Bank of Scotland assuming that a different amount to the reported fee has been recieved.
  4. Find the Bank of Scotland report from May 2023.

If you want to you can view these facilities:

  • Compressors Ltd
  • Petroleum Pipes Ltd
  • Marine Equipment Ltd
  • British Pharma Supplies Ltd

Start with the email notification or go straight to sign in (no sign in details needed).

Version 2

To complete the reconciliation task follow these steps:

  1. Use the Clear data link in the footer if you've just viewed version 1 or are starting this journey again.
  2. View this Excel document (opens on Sharepoint in a new tab) for payments received from Barclays.
  3. Start here (no sign in details needed) to begin the reconciliation process for Barclays with the amounts found on the 'Payments 1' sheet. All the other data that you need is in the prototype.
  4. Before continuing with the reconciliation task see if you can:
    • make changes to the date of a payment that you've already entered
    • find the Barclays report from May 2023
    • filter the 'Premium payments' tab to find the facilty with the facility ID 0020002167
  5. Now enter the final payment found on the 'Payments 2' sheet of the Excel document into the prototype.
  6. You can now assume that all payments have now been received – continue to do what you think you need to do to complete the reconciliation process.
